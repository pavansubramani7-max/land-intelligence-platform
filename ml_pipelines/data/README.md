# Sample Land Data

This directory contains the synthetic dataset used for training ML models.

## File: sample_land_data.csv

Generated by `ml_pipelines/scripts/generate_synthetic_data.py`

### Columns

| Column | Type | Description |
|--------|------|-------------|
| land_id | int | Unique identifier |
| location | str | City name |
| latitude | float | GPS latitude |
| longitude | float | GPS longitude |
| area_sqft | float | Land area in square feet |
| zone_type | str | residential/commercial/agricultural/industrial |
| road_proximity_km | float | Distance to nearest road (km) |
| infrastructure_score | float | 0-100 infrastructure quality score |
| year_established | int | Year land was registered |
| soil_type | str | loam/clay/sandy/rocky/alluvial |
| flood_risk | bool | Whether in flood zone |
| ownership_changes | int | Number of ownership transfers |
| litigation_count | int | Active litigations |
| market_price | float | Current market price (₹) |
| actual_value | float | True valuation (target for regression) |
| dispute_risk | str | low/medium/high (target for classification) |
| is_fraud | bool | Fraud label (target for anomaly detection) |
| historical_prices | json | List of 5 years of historical prices |

### Statistics (1000 rows)
- Cities: Mumbai, Delhi, Bangalore, Chennai, Hyderabad, Pune, Kolkata, Ahmedabad, Jaipur, Surat
- Zone distribution: ~50% residential, 25% commercial, 15% agricultural, 10% industrial
- Fraud rate: ~5%
- Dispute distribution: ~60% low, 30% medium, 10% high
